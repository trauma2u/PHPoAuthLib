PHPoAuthLib
===========
The Forked Project of [Lusitanian/PHPoAuthLib](https://github.com/Lusitanian/PHPoAuthLib) for Korean Services

Installation
------------

```json
{
    "repositories": [
        {
            "type": "git",
            "url": "https://github.com//trauma2u/PHPoAuthLib"
        }
    ],
    "require": {
        "lusitanian/oauth": "dev-master"
    }
}
```
